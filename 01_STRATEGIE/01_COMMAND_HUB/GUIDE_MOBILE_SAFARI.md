# 📱 GUIDE D'UTILISATION : IRIS TITAN MOBILE (SAFARI)

## 1. Installation sur iPhone (PWA)
Pour une expérience optimale, installez l'application sur votre écran d'accueil :

1.  Ouvrez **Safari** sur votre iPhone.
2.  Naviguez vers l'URL de production : `https://digital-flux.vercel.app` (ou votre URL personnalisée).
3.  Appuyez sur l'icône **Partager** (le carré avec une flèche vers le haut en bas de l'écran).
4.  Faites défiler vers le bas et sélectionnez **"Sur l'écran d'accueil"**.
5.  Appuyez sur **Ajouter**.

L'application apparaîtra désormais comme une application native avec son icône émeraude.

## 2. Utilisation Hors-Ligne
L'application est conçue pour fonctionner même sans connexion (ex: zones blanches à Tunis) :

-   **Données** : Les 103 prospects sont automatiquement mis en cache dès la première ouverture.
-   **Consultation** : Vous pouvez ouvrir l'app et consulter les profils tactiques sans internet.
-   **Modifications** : Si vous changez le statut d'un prospect alors que vous êtes hors-ligne, la modification est enregistrée dans le stockage local de votre téléphone.

## 3. Synchronisation Live
-   **Auto-Sync** : Dès que votre iPhone retrouve une connexion internet, l'application tentera de synchroniser les données avec le registre central.
-   **Sécurité** : Si vous fermez l'app, vous devrez vous ré-authentifier avec votre code souverain (`IRIS2026` par défaut).

---
*Note : Cette configuration garantit la continuité de vos opérations même si votre PC local est éteint.*
