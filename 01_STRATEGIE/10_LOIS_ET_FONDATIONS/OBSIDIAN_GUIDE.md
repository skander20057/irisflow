## 🏁 Configuration Rapide
1. **Ouvrir Obsidian.**
2. Cliquez sur **"Open folder as vault"**.
3. Sélectionnez le dossier racine : `/Users/hachicha/Desktop/digital flux`.
4. Autorisez l'indexation.

## 🗺️ Les Outils de Pilotage
- **Vue Globale** : Ouvre `01_STRATEGIE/FLUX_TACTIQUE.canvas`.
- **Vue Agents** : Ouvre `02_AGENTS/AGENTS_ORCHESTRATION.canvas`.
- **Journal Live** : Ouvre `01_STRATEGIE/00_PULSE_LIVE.md`.

## ⚡ Fonctionnement du "Live"
Chaque fois que tu consultes le Dashboard IRIS (ou que le système effectue une tâche), une ligne est ajoutée dans `00_PULSE_LIVE.md`. Dans Obsidian, tu verras ce fichier s'actualiser en temps réel.

3. **Advanced Tables** : Pour gérer ton CRM Markdown avec aisance.
